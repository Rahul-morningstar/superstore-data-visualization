# 📊 Superstore Data Visualization Project

## Overview
A comprehensive data analysis and visualization project on the 
Sample Superstore dataset (9,994 records), covering 30 
scenario-based business questions.

## Tools & Technologies
- Python (Matplotlib, Seaborn, Pandas)
- Tableau
- Excel (.xls)

## Dataset
- Records: 9,994
- Categories: Technology, Furniture, Office Supplies
- Years: 2019–2022
- Regions: West, East, Central, South

## Key Questions Explored
- Sales by category and region
- Monthly and quarterly trends
- Discount vs Profit analysis
- Customer segmentation
- Shipping mode performance
- Return rate analysis
- Sub-category profitability (Treemap)
...and 23 more!

## Files
| File | Description |
|------|-------------|
| `notebooks/Superstore_All30_Visualizations.ipynb` | Python notebook with all 30 charts |
| `tableau/Superstore_All30_Tableau.twb` | Tableau workbook |
| `output/Superstore_All30_Visualizations.pdf` | Exported PDF of all visualizations |
| `data/Sample_Superstore.xls` | Raw dataset |

## Key Insights
- Technology had the highest sales ($1.73M), followed by Furniture ($1.60M)
- West region contributed 31.7% of total sales
- Higher discounts (>30%) consistently led to losses
- Standard Class was the most used shipping mode (59.2%)
- Copiers were the most profitable sub-category
